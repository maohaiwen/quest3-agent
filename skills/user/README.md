# User Skills Directory

Put your custom skills here.

Each skill should be in its own directory with a `skill.md` file.

Example:

```
user_skills/
└── my_custom_skill/
    ├── skill.md
    └── main.py (optional)
```

## skill.md Format

```markdown
---
name: my_custom_skill
version: 1.0.0
description: Description of my skill
author: Your Name
tags: [tag1, tag2]
tools: [tool1, tool2]
---

# My Custom Skill

## Description
...
```
